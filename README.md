# 202277120-SultanAlsbia-assignment2

## Project Description
This project is my Assignment 2 interactive portfolio website for SWE363. It continues the portfolio from Assignment 1 and improves it with dynamic behavior, JavaScript-based user interaction, smoother transitions, and clearer user feedback.

## Main Features
- Theme switcher with saved user preference using `localStorage`
- Personalized greeting based on the visitor name and time of day
- Tab-based portfolio section switching
- Live project search with an empty-state message
- Contact form validation with success and error feedback
- Responsive design with hover effects and reveal animations

1. Clone or download the repository.
2. Open the folder in VS Code or another editor.
3. Run the project by opening `index.html` in a browser.
4. For easier development, you can use Live Server, but it is not required.
